# 정보앨범

후기와 정보 링크를 검색하고 필터링해서 보여주는 반응형 정적 웹사이트입니다. 백엔드, DB, 로그인, 글쓰기 기능 없이 HTML, CSS, JavaScript만 사용합니다.

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 디자인과 반응형 스타일
- `script.js`: 검색, 필터, 정렬, 카드 렌더링
- `data.json`: 후기/정보 링크 데이터
- `README.md`: 실행 방법

## 실행 방법

가장 간단한 방법은 `index.html` 파일을 브라우저로 여는 것입니다.

일부 브라우저는 보안 정책 때문에 로컬 파일 상태에서 `data.json` 읽기를 막을 수 있습니다. 이 경우 화면에는 내장 예시 데이터가 표시됩니다. GitHub Pages, Netlify, Vercel 같은 정적 호스팅에 올리면 `data.json`을 자동으로 불러옵니다.

## 데이터 추가 방법

`data.json` 배열에 아래 형식의 객체를 추가하면 됩니다. 정적 호스팅에 배포된 상태에서는 새로고침만으로 웹에 반영됩니다.

```json
{
  "id": 6,
  "title": "새 후기 제목",
  "category": "카테고리명",
  "tags": ["태그1", "태그2"],
  "summary": "후기 요약 문장",
  "date": "2024-06-01",
  "url": "https://example.com"
}
```

## 배포 방법

이 폴더의 파일 5개를 그대로 정적 호스팅 서비스에 업로드하세요. 별도 빌드 명령이나 서버 설정은 필요하지 않습니다.

## 주요 기능

- 제목, 요약, 태그, 카테고리 실시간 검색
- 카테고리 필터
- 태그 필터
- 최신순 / 오래된순 정렬
- 검색 결과 개수 표시
- 결과 없음 안내 문구
- 모바일 하단 네비게이션 느낌의 필터 영역
